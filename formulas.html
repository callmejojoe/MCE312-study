📡 Module 1 — Signals & Systems

Elementary Signals

Unit Step
u(t) = 1 for t≥0, 0 for t<0
Ramp
r(t) = t·u(t)
Sinusoid
x(t) = A·sin(ωt + φ), ω = 2πf
Impulse Sifting
∫x(t)·δ(t−t₀)dt = x(t₀)
Relation
δ(t) = d/dt[u(t)] u(t) = ∫₋∞ᵗ δ(τ)dτ

Signal Classifications

Even Signal
x(t) = x(−t)
Odd Signal
x(t) = −x(−t)
Periodic
x(t) = x(t+T), f = 1/T
Energy Signal (E finite)
E = ∫₋∞^∞ |x(t)|² dt, 0<E<∞, P=0
Power Signal (P finite)
P = lim(1/2T)∫₋ᵀᵀ|x(t)|²dt, 0<P<∞, E=∞

Time Operations

Shift Right (Delay)
y(t) = x(t − t₀) [shifts signal right by t₀]
Shift Left (Advance)
y(t) = x(t + t₀) [shifts signal left by t₀]
Time Scaling
y(t) = x(At) |A| > 1 → Compression (faster) |A| < 1 → Expansion (slower)
Time Reversal
y(t) = x(−t) [flip about t=0]

System Properties

Linearity (Superposition)
ax₁(t)+bx₂(t) → ay₁(t)+by₂(t)
Time Invariance Test
x(t−t₀) → y(t−t₀) for all t₀
Causality
y(t) depends only on x(τ) for τ ≤ t
BIBO Stability (LTI)
∫₋∞^∞ |h(t)| dt < ∞
LTI Convolution
y(t) = x(t) * h(t) = ∫x(τ)h(t−τ)dτ

🔢 Module 2 — Sampling & Quantisation

Sampling

Sampling Period & Frequency
Ts = 1/fs
Nyquist Criterion (No Aliasing)
fs ≥ 2·fmax (Nyquist Rate = 2·fmax)
Nyquist Frequency
fN = fs / 2 (max representable frequency)

Quantisation

Number of Levels
L = 2^W (W = word length in bits)
Step Size (Resolution)
Δ = V_range / (2^W − 1)
Max Quantisation Error
|e_q|_max = Δ/2
Dynamic Range
DR ≈ 6.02 × W dB (~6 dB/bit)

🌊 Module 3 — Fourier Transforms

Continuous Fourier Transform

Forward FT (Analysis)
X(ω) = ∫₋∞^∞ x(t)·e^(−jωt) dt
Inverse FT (Synthesis)
x(t) = (1/2π)∫₋∞^∞ X(ω)·e^(jωt) dω
Frequency Variable
ω = 2πf (rad/s) f = ω/2π (Hz)

Key FT Pairs

Impulse
δ(t) ↔ 1
Constant
1 ↔ 2π·δ(ω)
Complex Exponential
e^(jω₀t) ↔ 2π·δ(ω−ω₀)
Rect Pulse (width τ)
rect(t/τ) ↔ τ·sinc(ωτ/2π)

FT Properties

Linearity
ax(t)+by(t) ↔ aX(ω)+bY(ω)
Time Shift
x(t−t₀) ↔ e^(−jωt₀)·X(ω)
Frequency Shift
e^(jω₀t)·x(t) ↔ X(ω−ω₀)
Convolution Theorem
x(t)*h(t) ↔ X(ω)·H(ω)
Differentiation
dx/dt ↔ jω·X(ω)
Parseval's Theorem
∫|x(t)|²dt = (1/2π)∫|X(ω)|²dω

Discrete Fourier Transform (DFT)

Forward DFT (Analysis)
X(k) = Σₙ₌₀^(N−1) x(n)·e^(−j2πkn/N) k = 0, 1, ..., N−1
Inverse DFT (Synthesis)
x(n) = (1/N)Σₖ₌₀^(N−1) X(k)·e^(j2πkn/N) n = 0, 1, ..., N−1
Twiddle Factor
W_N = e^(−j2π/N) Powers: W_N^0=1, W_N^(N/4)=−j, W_N^(N/2)=−1
DC Component
X(0) = Σₙ x(n) (sum of all samples)
Frequency Resolution
Δf = fs / N fₖ = k·(fs/N)
Parseval for DFT
Σ|x(n)|² = (1/N)·Σ|X(k)|²

⚡ Exam Quick Tips

🎯 Test Linearity

Check BOTH: (1) Additivity: x₁+x₂ → y₁+y₂, and (2) Homogeneity: cx → cy. Fail either → nonlinear.

⏱ Time Shift Direction

x(t − t₀): shift RIGHT (delay). x(t + t₀): shift LEFT (advance). The sign inside the bracket flips your intuition!

📊 DFT DC Component

X(0) is always the sum of all samples. Quick sanity check for your DFT calculations.

🔄 Nyquist Gotcha

fs ≥ 2·fmax is the MINIMUM. Sampling at exactly 2·fmax is theoretically sufficient but practically risky due to non-ideal filters.

⚠️ Causality vs Stability

Causal: h(t)=0 for t<0. Stable: ∫|h(t)|dt<∞. A system can be one without the other!

🌊 FT Sign Convention

Forward FT: e^(−jωt). Inverse FT: e^(+jωt). Memorize: "Forward is Negative, Inverse is Positive."

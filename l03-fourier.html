Fourier Transforms

Transforming signals from the time domain into the frequency domain — the mathematical language of signal processing. Covers both the Continuous Fourier Transform (CFT) and the Discrete Fourier Transform (DFT).

📡 Frequency Domain
🔢 CFT + DFT
🧪 10-Q Quiz Below

1. Why Fourier Transforms?

Motivation

From Time Domain to Frequency Domain

The Fourier Series can represent any periodic signal as a sum of sinusoids (complex exponentials). But many real-world signals — a single pulse, a transient sound, a noise burst — are aperiodic (non-periodic). The Fourier Transform extends the Fourier Series to handle aperiodic signals by letting the period T → ∞.

What the FT Reveals

  • The frequency content (spectrum) of a signal
  • Which frequencies carry the most energy
  • How to design filters that pass/block specific frequencies
  • The bandwidth requirements for transmission

Time ↔ Frequency Duality

2. Continuous Fourier Transform (CFT)

Core Formula

⭐ The Fourier Transform Pair

The Fourier Transform decomposes a time-domain signal x(t) into its frequency components X(ω):

X(ω) = ℱ{x(t)} = ∫₋∞^∞ x(t) · e^(−jωt) dt Where: ω = 2πf (angular frequency, rad/s) j = √(−1) (imaginary unit)
x(t) = ℱ⁻¹{X(ω)} = (1/2π) ∫₋∞^∞ X(ω) · e^(jωt) dω

X(ω) is generally complex-valued: X(ω) = |X(ω)|·e^(j∠X(ω)) where |X(ω)| is the magnitude spectrum and ∠X(ω) is the phase spectrum.

💡

Intuition: The Fourier Transform is like a "frequency microscope." It tells you: out of all possible sinusoids (at every frequency), how much of each frequency is present in your signal x(t)? The answer for each ω is X(ω).

⚠️ The Convolution-Multiplication Duality

This is arguably the most important FT property for engineers: convolution in the time domain becomes multiplication in the frequency domain. This is why we use Fourier Transforms to analyze LTI systems — instead of computing a convolution integral (hard), we just multiply spectra (easy).

y(t) = x(t) * h(t) ←→ Y(ω) = X(ω) · H(ω) H(ω) = Y(ω)/X(ω) is the Transfer Function (Frequency Response)

4. Worked Examples — Continuous FT

Calculation

📘 Example 1: FT of a Rectangular Pulse

Find the Fourier Transform of x(t) = rect(t/τ) — a rectangle of width τ centered at t=0.

x(t) = { 1, |t| ≤ τ/2 { 0, |t| > τ/2
X(ω) = ∫₋τ/2^(τ/2) 1 · e^(−jωt) dt = [e^(−jωt)/(−jω)]₋τ/2^(τ/2) = (e^(−jωτ/2) − e^(jωτ/2)) / (−jω) = (2 sin(ωτ/2)) / ω = τ · sinc(ωτ/2π) [using sinc(x) = sin(πx)/(πx)]
💡

Key insight: A rectangular pulse in time ↔ a sinc function in frequency. A wider pulse (larger τ) → a narrower sinc (less bandwidth). This is the time-bandwidth product uncertainty principle.

📘 Example 2: FT of the Unit Impulse

x(t) = δ(t) X(ω) = ∫₋∞^∞ δ(t) · e^(−jωt) dt = e^(−jω·0) [by the sifting property] = 1 ∴ ℱ{δ(t)} = 1 Interpretation: The impulse contains ALL frequencies equally. An impulse in time → a flat (constant) spectrum.

This is why the impulse response h(t) of an LTI system fully characterizes it — its FT H(ω) reveals the system's entire frequency response.

5. Discrete Fourier Transform (DFT)

Digital Processing

From Continuous to Discrete

In practice, signals are finite-length discrete-time sequences (stored in memory). The DFT operates on a block of N samples and produces N complex frequency coefficients. It is the computational engine behind all digital signal processing.

⭐ DFT Definition

X(k) = Σₙ₌₀^(N−1) x(n) · e^(−j2πkn/N) k = 0, 1, ..., N−1 Compact notation: X(k) = Σₙ₌₀^(N−1) x(n) · Wₙ^(kn) Where the twiddle factor: Wₙ = e^(−j2π/N)
x(n) = (1/N) · Σₖ₌₀^(N−1) X(k) · e^(j2πkn/N) n = 0, 1, ..., N−1
💡

Key point: The DFT is periodic with period N. X(k) represents the signal's content at frequency f = k·(fs/N). The frequency resolution is fs/N — to get finer frequency resolution, either increase N (use more samples) or decrease fs.

6. DFT Worked Examples

Calculation

📘 Example: 4-point DFT of x(n) = {1, −1, 2, −2}

N = 4. Compute X(k) for k = 0, 1, 2, 3 using X(k) = Σₙ₌₀³ x(n)·e^(−j2πkn/4)

X(0) = x(0)·W₄⁰ + x(1)·W₄⁰ + x(2)·W₄⁰ + x(3)·W₄⁰ = 1(1) + (−1)(1) + 2(1) + (−2)(1) = 1 − 1 + 2 − 2 = 0
W₄ = e^(−j2π/4) = e^(−jπ/2) = −j X(1) = x(0)·1 + x(1)·W₄¹ + x(2)·W₄² + x(3)·W₄³ = 1(1) + (−1)(−j) + 2(−1) + (−2)(j) = 1 + j − 2 − 2j = −1 − j
W₄² = e^(−jπ) = −1 X(2) = x(0)·1 + x(1)·(−1) + x(2)·1 + x(3)·(−1) = 1 + 1 + 2 + 2 = 6
W₄³ = e^(−j3π/2) = j X(3) = x(0)·1 + x(1)·j + x(2)·(−1) + x(3)·(−j) = 1 + (−1)(j) + 2(−1) + (−2)(−j) = 1 − j − 2 + 2j = −1 + j Result: X(k) = {0, −1−j, 6, −1+j}

📘 Example: 4-point IDFT of X(k) = {4, 2, 0, 4}

x(n) = (1/4) Σₖ₌₀³ X(k) · e^(j2πkn/4) n=0: x(0) = (1/4)[4(1) + 2(1) + 0(1) + 4(1)] = (1/4)(10) = 2.5 n=1: x(1) = (1/4)[4 + 2j + 0 + 4(j)] = (1/4)[4 + 6j] = 1 + 1.5j ← Note: real sequences have conjugate-symmetric DFTs n=2: x(2) = (1/4)[4 + 2(−1) + 0 + 4(1)] = (1/4)(6) = 1.5 n=3: x(3) = (1/4)[4 + 2(−j) + 0 + 4(−j)] = (1/4)[4 − 6j] = 1 − 1.5j Note: If X(k) doesn't satisfy conjugate symmetry, x(n) will be complex.

7. Key DFT Properties

Reference

Periodicity

Both x(n) and X(k) are periodic with period N:

X(k+N) = X(k) x(n+N) = x(n)

Symmetry (Real x(n))

If x(n) is a real sequence, the DFT is conjugate symmetric:

X(N−k) = X*(k) |X(N−k)| = |X(k)| ∠X(N−k) = −∠X(k)

Circular Convolution

DFT multiplication in frequency ↔ circular convolution in time:

x(n) ⊛ h(n) ↔ X(k)·H(k)

This differs from linear convolution. Zero-padding the sequences before DFT converts circular to linear convolution.

Energy (Parseval's)

Σₙ₌₀^(N−1) |x(n)|² = (1/N) Σₖ₌₀^(N−1) |X(k)|²

Total energy computed in time domain equals (scaled) total energy computed from DFT coefficients.

Frequency Resolution of the DFT

Δf = fs / N Where: fs = sampling frequency N = number of DFT points (block length) k-th DFT bin → frequency fₖ = k · (fs/N) To improve frequency resolution: → Increase N (use more samples / zero-pad) → Or decrease fs (sample more slowly — but watch aliasing!)

Question 02 / 10
Convolution in the time domain corresponds to what operation in the frequency domain?
A is correct: This is the Convolution Theorem — one of the most powerful results in signal processing: x(t)*h(t) ↔ X(ω)·H(ω). It converts the computationally expensive convolution integral into simple pointwise multiplication. This is why filtering is implemented via FFT in practice.
• B: Addition in time → addition in frequency (linearity), but convolution → multiplication.
• C and D are incorrect — convolution does not remain convolution in the frequency domain.
Question 03 / 10
The Fourier Transform of the unit impulse δ(t) is:
C is correct: ℱ{δ(t)} = ∫δ(t)·e^(−jωt)dt = e^(−jω·0) = 1. The impulse has a flat, uniform spectrum containing all frequencies with equal amplitude. This is why "impulsive" events (explosions, lightning) create broadband noise.
• A (δ(ω)) is the FT of a constant x(t)=1, which is the dual relationship.
• B (jω) is the FT of the derivative operator.
• D is the FT of δ(t−1), a time-shifted impulse.
Question 04 / 10
The N-point DFT of a sequence x(n) is defined by X(k) = ?
D is correct: The DFT forward transform is X(k) = Σₙ₌₀^(N−1) x(n)·e^(−j2πkn/N). The (1/N) factor belongs to the INVERSE DFT. The negative sign in the exponent belongs to the forward (analysis) transform.
• A has the wrong sign (positive → inverse) AND the 1/N factor (inverse).
• B has the correct factor (no 1/N) but wrong sign.
• C has the correct sign but the 1/N factor — that's closer to the inverse DFT form.
Question 05 / 10
The twiddle factor WN used in the DFT is equal to:
B is correct: W_N = e^(−j2π/N). This places it on the unit circle in the complex plane, rotating clockwise by 2π/N radians per increment. Its powers W_N^(kn) trace out N equally-spaced complex exponentials that form the DFT basis functions.
• A has the positive sign — that would be the inverse DFT twiddle factor W_N*.
• C has the wrong angle (π instead of 2π).
• D is only the real part — it ignores the imaginary component.
Question 06 / 10
A time shift of t₀ in a signal x(t) → x(t−t₀) corresponds to what change in the Fourier Transform X(ω)?
C is correct: Time shifting multiplies the spectrum by a complex exponential e^(−jωt₀). Critically, this multiplier has magnitude 1 for all ω — so the magnitude spectrum |X(ω)| is unchanged. Only the phase is affected: ∠X(ω) decreases by ωt₀ radians. This makes sense: shifting a signal in time shouldn't change "how much" of each frequency is present.
• A is frequency shifting (modulation theorem).
• B has no basis.
• D is the time-reversal property.
Question 07 / 10
Parseval's theorem for the DFT states that:
A is correct: Parseval's theorem: Σ|x(n)|² = (1/N)Σ|X(k)|². Energy is conserved between domains (with the 1/N normalization factor). This is fundamental to spectral analysis — power computed in time must equal power computed from the spectrum.
• B is false — X(k) is generally complex even if x(n) is real.
• C is false — DFT is periodic with period N, not 2N.
• D is the symmetry property for real signals, not Parseval's.
Question 08 / 10
To improve the frequency resolution of a DFT, one should:
D is correct: Frequency resolution Δf = fs/N. To make Δf smaller (finer resolution), increase N. You can do this by: (1) recording more data samples, or (2) zero-padding the existing samples.
• A: Increasing fs widens the frequency range covered, but actually worsens resolution (Δf = fs/N increases if fs increases with fixed N).
• B: Decreasing N worsens resolution (fewer, wider bins).
• C: Time reversal doesn't affect frequency resolution.
Question 09 / 10
Compute X(0) (the DC component) of the 4-point DFT of x(n) = {1, 2, 3, 4}.
B is correct: X(0) = Σₙ₌₀^(N−1) x(n)·e^0 = Σ x(n) · 1 = 1+2+3+4 = 10. The DC component (k=0) is always the sum of all samples, because e^0 = 1.
• A is just x(0).
• C is x(3) or N.
• D = X(0)/N = 10/4 = 2.5 — this would be the mean (average), not the DFT DC term.
Question 10 / 10
The main difference between the Continuous Fourier Transform (CFT) and the Discrete Fourier Transform (DFT) is:
C is correct: The CFT handles continuous-time, infinite-duration signals and produces a continuous frequency spectrum X(ω). The DFT handles finite-length, discrete-time sequences x(n) (N samples) and produces exactly N discrete frequency coefficients X(k). The DFT is computationally tractable and implemented efficiently via the FFT algorithm.
• A is false — both use complex exponentials (Euler's formula), combining both sine and cosine.
• B is backwards — the Fourier Series handles periodic signals; the CFT extends this to aperiodic signals by letting T→∞.
• D: Both are related to Nyquist when considering sampling — the distinction isn't about the Nyquist theorem.

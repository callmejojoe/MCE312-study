📡 Module 01

Introduction to Signals & Systems

The foundational vocabulary of signal processing — what signals and systems are, how we classify them, and the basic mathematical operations we perform on them.

🎯 8 Core Topics
📐 Signal Operations
🧪 10-Q Quiz Below

1. What Is a Signal?

Core Concept

Definition

A signal is a function of one or more independent variables that conveys information about a physical phenomenon. Most commonly, signals are functions of time, but they can also be functions of space, frequency, or any other variable of interest.

  • Voice signal: air pressure as a function of time.
  • ECG: electrical heart activity vs. time.
  • Image: pixel intensity as a function of 2D space.
  • Video: intensity as a function of both space and time.
💡

Intuition: Think of a signal as the "data" that carries a message. The signal itself doesn't do anything — it just represents information. Systems are what act on signals.

2. Elementary (Basic) Signals

Building Blocks

Why Elementary Signals Matter

Elementary signals are the "atoms" of signal analysis. Any complex real-world signal can be built from combinations of these. They also act as standard test inputs for characterizing system behavior.

Unit Step Signal — u(t)

Turns ON at t = 0 and stays ON. Used to model switching events.

u(t) = { 1, t ≥ 0 { 0, t < 0

Unit Impulse Signal — δ(t)

An infinitely narrow, infinitely tall spike at t = 0 with area = 1. It is the derivative of the unit step.

∫ x(t)·δ(t−t₀) dt = x(t₀)

Ramp Signal — r(t)

r(t) = { t, t ≥ 0 { 0, t < 0 Note: r(t) = t·u(t) Also: u(t) = d/dt[r(t)]

The integral of the step function. Models linearly increasing quantities (e.g., constant-velocity motion).

Sinusoidal Signal

x(t) = A·sin(ωt + φ) A = Amplitude ω = Angular frequency (rad/s) φ = Phase (radians) T = 2π/ω = Period

The most important signal in engineering. Foundation of Fourier analysis — every periodic signal decomposes into sinusoids.

3. Classification of Signals

Taxonomy
Classification Type A Type B Key Test
Time domain Continuous-Time (CT) — defined for all t Discrete-Time (DT) — defined only at t = nTs Is t continuous or sampled?
Predictability Deterministic — fully described by equation Random (Non-deterministic) — probabilistic Can we write an exact formula?
Symmetry Even: x(t) = x(−t) Odd: x(t) = −x(−t) Mirror about t = 0
Repetition Periodic: x(t) = x(t+T) ∀t Aperiodic: No repeating pattern Does it repeat exactly?
Energy Energy signal: 0 < E < ∞, P = 0 Power signal: 0 < P < ∞, E = ∞ Compute E and P

⚡ Energy vs Power Signals

E = ∫₋∞^∞ |x(t)|² dt
P = lim(T→∞) (1/2T) ∫₋ᵀᵀ |x(t)|² dt

A signal cannot be both energy AND power (with non-zero finite values). A periodic signal is always a power signal.

Even/Odd Decomposition

Any signal can be split into even and odd parts:

xₑ(t) = [x(t) + x(−t)] / 2 ← Even part xₒ(t) = [x(t) − x(−t)] / 2 ← Odd part x(t) = xₑ(t) + xₒ(t)

Example: sin(t) is purely odd. cos(t) is purely even.

4. Basic Operations on Signals

Operations

The Two Axes: Amplitude and Time

All signal operations modify either the amplitude (y-axis) or the time (x-axis) of a signal. Understanding this distinction is critical for exam questions.

Amplitude Operations

  • Scaling: y(t) = C·x(t) — stretches/compresses vertically
  • Addition: y(t) = x₁(t) + x₂(t) — add amplitudes point-by-point
  • Multiplication: y(t) = x₁(t)·x₂(t) — multiply amplitudes point-by-point

Time Operations

  • Shifting: x(t − t₀) shifts RIGHT by t₀ (delay); x(t + t₀) shifts LEFT (advance)
  • Scaling: x(At) — if |A|>1: compress; if |A|<1: expand
  • Reversal: x(−t) — flips signal about the vertical axis

5. What Is a System?

Systems

Definition

A system is any process or device that takes an input signal (excitation) and produces an output signal (response). It transforms signals.

6. Classification of Systems

Critical for Exam

⭐ LTI Systems — The Most Important Class

A Linear Time-Invariant (LTI) system is both linear and time-invariant. LTI systems are the foundation of signal processing because they can be fully characterized by their impulse response h(t), and the output is simply the convolution of input with h(t):

y(t) = x(t) * h(t) = ∫₋∞^∞ x(τ)·h(t−τ) dτ

Linearity

A system is linear if it satisfies superposition:

If x₁ → y₁ and x₂ → y₂, then: a·x₁ + b·x₂ → a·y₁ + b·y₂

Test: If y = x² → NOT linear (fails homogeneity). If y = 3x → linear.

Time Invariance

Output shifts in time exactly like the input — the system doesn't change with time.

If x(t) → y(t), then: x(t − t₀) → y(t − t₀) for all t₀

Test: Replace t with (t−t₀) in system equation. If result matches y(t−t₀), it's TI.

Causality

A causal system's output depends only on present and past inputs — not future inputs. Physically realizable real-time systems must be causal.

y(t) = x(t) + x(t−1) ← Causal ✓ y(t) = x(t) + x(t+1) ← Non-causal ✗

BIBO Stability

A system is stable (Bounded-Input Bounded-Output) if every bounded input produces a bounded output.

∫₋∞^∞ |h(t)| dt < ∞ (absolutely integrable)

An unstable system: y(t) = e^t·x(t) (unbounded even for x(t)=1).

Static vs Dynamic

Static (memoryless): output at time t depends only on input at time t.

Dynamic: output depends on past (or future) values — has memory.

Invertible Systems

A system is invertible if different inputs always produce different outputs (one-to-one mapping). There exists an inverse system H⁻¹ such that H⁻¹{H{x}} = x.

LTV vs LTI

LTV: Linear + Time-Variant (coefficients change with time).

LTI: Linear + Time-Invariant — the gold standard; analyzed via impulse response and Fourier/Laplace transforms.

⚠️

Exam Trap: When testing linearity, always check BOTH additivity (x₁+x₂ → y₁+y₂) AND homogeneity (a·x → a·y). A system like y = |x| passes additivity tests but fails homogeneity for a = −1. Always check both conditions.

Question 01 / 10
Which of the following is the correct mathematical definition of an even signal?
C is correct: An even signal is symmetric about the vertical axis (t=0), meaning x(t) = x(−t).
• A describes an odd signal.
• B describes a periodic signal with period T.
• D describes a time-shifted signal, not symmetry.
Question 02 / 10
The unit impulse signal δ(t) is best described as the __________ of the unit step signal u(t).
B is correct: δ(t) = du(t)/dt. Conversely, u(t) = ∫₋∞ᵗ δ(τ)dτ. The impulse is the "instantaneous transition" captured as a derivative of the step jump at t=0.
• A is wrong — the integral of δ(t) gives u(t), not the other way around. C and D are mathematically incorrect relationships.
Question 03 / 10
Signal x(t) is shifted to produce y(t) = x(t + 3). This operation represents a:
D is correct: x(t + t₀) shifts the signal LEFT (advances it) by t₀. Think of it as: the signal reaches any given amplitude value earlier in time.
• A is wrong — x(t − t₀) gives a right shift (delay).
• B is x(−t).
• C is x(3t), not x(t+3).
Question 04 / 10
Which condition must hold for a signal to be classified as periodic?
A is correct: A periodic signal repeats exactly every T seconds for ALL time — both past and future. T is the fundamental period and 1/T is the fundamental frequency.
• B is the even signal condition.
• C is the energy signal condition (finite energy).
• D describes signals that decay to zero (not periodic).
Question 05 / 10
A system described by y(t) = x²(t) is tested for linearity. What is the correct conclusion?
C is correct: Test homogeneity: if x → y=x², then a·x → (ax)² = a²x² ≠ a·x². Since a²x² ≠ a·x² for general a, homogeneity fails, so the system is nonlinear.
• A and B are incorrect — direct relationship or partial linearity doesn't make a system linear.
• D is a property of invertibility, not linearity.
Question 06 / 10
A system y(t) = x(t − 2) + x(t + 1) is tested for causality. The correct verdict is:
B is correct: x(t+1) means the output at time t depends on the input at time t+1, which is a future value. A causal system may only use present (t) and past (t−τ, τ>0) values. x(t−2) is fine (past), but x(t+1) makes the system non-causal.
• Time invariance and number of terms are irrelevant to causality.
Question 07 / 10
For a CT signal, when is it classified as an energy signal?
A is correct: An energy signal has finite, non-zero total energy (E < ∞), and because it "dies out," its time-averaged power is zero.
• B describes a power signal.
• C is impossible — if E is finite, P = lim(1/2T)·E → 0 as T→∞; they can't both be nonzero and finite simultaneously.
• D: Periodic signals are power signals (infinite energy, finite average power).
Question 08 / 10
Time scaling x(2t) applied to a signal results in:
D is correct: When |A|>1 in x(At), the signal is compressed in time (it runs faster). x(2t) completes in half the time of x(t). To expand (slow down), use |A|<1, e.g. x(0.5t).
• A is the opposite — that would be x(0.5t).
• B is time shifting, not scaling.
• C: Amplitude is unchanged by time scaling.
Question 09 / 10
What is the stability condition for an LTI system in terms of its impulse response h(t)?
B is correct: BIBO stability for LTI systems requires that the impulse response is absolutely integrable. If this holds, any bounded input |x(t)| ≤ M produces a bounded output |y(t)| ≤ M·∫|h(t)|dt < ∞.
• A is the causality condition (h(t)=0 for t<0 means causal).
• C and D have no relation to stability.
Question 10 / 10
A random noise signal that cannot be described by a mathematical formula is best classified as:
C is correct: A signal that cannot be described by an exact mathematical formula and has uncertainty in its value is called non-deterministic or random. Such signals are modeled probabilistically using statistics (mean, variance, power spectral density).
• A and B require a mathematical formula — that's the definition of deterministic.
• D describes a signal type in the time domain and energy domain, which is independent of determinism.
